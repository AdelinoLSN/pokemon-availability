# pokemon-availability

## Notes about the data

- Forms are stored in the same file as their base Pokémon; the only difference is the form name. For example, Raichu and Raichu-Alola are included in the same file, but each has a different form value. (All .json files contain arrays of Pokémon, so a single file may include multiple Pokémon for future expansion.)

- Event Pokémon are generally not included, except in cases where the event allows the player to catch the Pokémon. For example, Mew is included in RSE because it can be caught in Emerald using the Old Sea Map, but it is not included in FRLG, where it can only be obtained through a distribution event.
